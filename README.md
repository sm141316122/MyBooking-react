# MyBooking網站 (React 使用Next.js框架) (後端開發中)
此專案以模仿Booking訂房網站作為練習，前端使用React並以Next.js為框架，以Axios串接後端。後端會使用Express、MondoDB來完成RESTFul API與資料庫，目前後端仍在開發中。
